# AWS Lambda with DynamoDB

## Requirements

- Docker

## Setup

### Start local stack
```
docker-compose up -d
```

### Connect and setup

1. connect into local environment
```
docker-compose exec lambda bash
```

2. Install dependences
```
$ npm install
```

### Start application
```
$ npm start
```

### Stop Application
```
$ npm stop
```

### Restart Application
```
$ npm restart
```

### Exit from Container
```
$ exit
```

### Stop local stack
```
docker-compose down
```
